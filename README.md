# Chatter App

## Description

Chatter App for random News from the internet with Web Scrapping using React.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# production mode
$ npm run build
```

## Test

```bash
# unit tests
$ npm run test
```
